# Sleep Tight

A high-polish, offline-first Android game about protecting a sleeping baby from bothersome flies.

## Game Overview
- **Art Style**: Stylized 3D (Pixar-inspired).
- **Core Loop**: Protect the baby from flies by Shooing (Swipe) or Squashing (Tap).
- **Platform**: Android.
- **Privacy**: 100% Offline, no data sharing.
